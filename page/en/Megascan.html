- Megascan -

Project Context

The project involves creating an environment using Megascan in Unreal Engine with the help of Quixel.

Our Project

The goal was to create a realistic environment by using Quixel Megascan to download assets and Quixel Bridge to import them.
This project took one week and allowed me to explore Unreal Engine 5.0.

Project Creation Date

This project was completed during the first year of apprenticeship at Pole Prod while studying in the fourth year at Gaming Campus, specializing in computer science with a focus on video games.
