- Cowpocalypse -

CONTEXT

Le projet consiste à créer un jeu de gestion de ressources sur Unity. Le temps imparti est de 1 semaine. Le langage utilisé est le C#.

Notre jeu

Industrialisez vos terres dans un Factorio-like et atteignez votre objectif : créer une armée de Vache Robot afin de dominer le monde ! Oui, rien de mieux que des vaches pour faire peur… Gérez vos ressources grâce aux composants disponibles permettant de : Récolter, Combiner, Regrouper, Dispersez, Déplacer et Vendre vos ressources. La map est générée de façon procédurale.

Choix d'architecture

Utilisation de :
  • Unity : pour déployer sur mobile également
  • Singleton : pour rendre les managers facilement accessibles
  • Object Pool : optimiser la gestion des ressources
  • Observer : découpler le son et le gameplay
  • State : distinguer les différentes phases du jeu
  • Scriptable object : réunir toutes les données du jeu au même endroit
  • Tâche asynchrones :sauvegarder et charger la partie sans interruption
